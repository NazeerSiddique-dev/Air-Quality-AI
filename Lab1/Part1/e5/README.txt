Exercise 5: MongoDB Operations using Python

Files:
mongodb_operations.py -> Python program performing MongoDB CRUD operations

Requirements:
Install PyMongo:
pip install pymongo

MongoDB must be installed and running locally.

Run program:
python mongodb_operations.py

Operations Demonstrated:
1. Connect to MongoDB
2. Create database
3. Create collection
4. Insert documents
5. Read documents
6. Update document
7. Delete document
